# TermKit

A Model-View-Controller Framework for Terminal applications.

## Install

The preferred method of installation is via RubyGems.org:  
<https://rubygems.org/gems/termkit>

	gem install termkit

or via `Gemfile`:

	gem 'termkit', '~>0.0'
